Sonos Player: The available or already found Sonos Player are listed here. The room name etc. as well as other network-related data can not be changed. In addition the 3 default volume values ​​for T2S announcement, Sonos standard volume and maximum volume per player must be specified. These are used as the default volume, if no syntax "volume" or "groupvolume" is specified in the syntax.
Please refer to the Wiki for more information. If a new Sonos player has been added to your existing system, please press the Scan Zones button to add them to your configuration.

Text-to-speech Instance: Select your Text-2-Speech instance to be used.

API Key: Necessary key to use for VoiceRRS, Ivona or Polly.

Secret Key: Necessary key to use for Ivona or Polly.

Voice: Select your Voice (only German Users right now).

Timeperiod to store MP3 files: Determines how many days a T2S file should be stored in tts folder before files will be deleted automatically.

Filename for Jingle : Name of the file which can be played as a kind of jingle before the announcement take place. The file MUST be stored in the subfolder loxberry/data/plugins/sonos4lox/tts/MP3.

Connection to Loxone: Switches the data transmission to Loxone On or Off. The Zone /Play/Pause/Stop buttons can be used for each zone in a Playlist Title/Artist or Radio, the current volume, and the Zone status. You can find more information about this and the Loxone Config in the Wiki.

UDP Port for Loxone: Port for data transfer. Play/Stop/Pause and Volume are transferred via UDP packet, title/artist/radio station to virtual text input connector, but only if the corresponding zone is currently running!

Debugging: Only activate if detailed error search is to be carried out in the browser.

Change Volume: Value by which the volume should be increased/decreased. This function is mainly used as a push button function.

Rampto: Function to decrease / increase the volume slowly. The following options are available:
sleep: Increases the volume within 17 seconds to desired volume alarm: Increases / decreases the volume within a few seconds (recommended) auto: Increases the volume within seconds (fast and steady)

Rampto Volume: Volume in percent from when 'rampto' WITHOUT explicit data is to be used, otherwise it goes on with the adjusted volume This just applies for the following functions: play, nextradio, playqueue, sonosplaylist, radioplaylist.

Radio Favorites: Here the radio favorites can be entered, which are then by-buttoned by button click. When the last station is reached, it goes backwards.
How to determine the relevant data for the radiosender please refer to the Wiki.
